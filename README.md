# Zhihu Pure Reader (知乎纯净阅读器)

**Make Zhihu Readable Again.**  
让知乎回归阅读本质。这是一个 Chrome 浏览器扩展，用于移除知乎页面上的广告、侧边栏和干扰元素，提供类似 Kindle 或杂志的沉浸式阅读体验。

![Icon](icon.png)

## ✨ 主要功能

*   **🛡️ 极简净化**：移除顶部导航栏、右侧推荐栏、底部悬浮广告、视频推荐等一切干扰阅读的元素。
*   **🎨 多样主题**：内置 4 种精心调配的阅读主题，适应不同环境。
    *   🌞 **暖色模式** (默认)：类似 Solarized Light，护眼舒适。
    *   🌙 **夜间模式**：深色背景，夜晚阅读不刺眼。
    *   📄 **纯白模式**：高对比度，类似纸张体验。
    *   📖 **羊皮模式**：复古质感。
*   **🎛️ 高度可定制**：
    *   **字体切换**：一键切换 宋体 (衬线) / 黑体 (无衬线)。
    *   **字号调节**：拖动滑块实时调整正文字号。
    *   **宽度调节**：自定义阅读区域宽度，适应不同屏幕。
*   **🧠 智能激活**：仅在“问题页”和“文章页”自动进入阅读模式，主页和个人页保持原样，方便导航。

## 📥 如何安装 (开发者模式)

在插件正式上架 Chrome Web Store 之前，你可以通过以下步骤手动安装：

1.  **下载代码**：
    *   点击本页面右上角的 `Code` -> `Download ZIP`，下载并解压到你的电脑上（记住解压后的文件夹路径）。
    *   或者使用 Git 克隆：`git clone https://github.com/your-repo/ZhihuRead.git`

2.  **打开 Chrome 扩展管理页**：
    *   在浏览器地址栏输入：`chrome://extensions/` 并回车。

3.  **开启开发者模式**：
    *   在页面右上角，找到 **"开发者模式" (Developer mode)** 开关并打开。

4.  **加载插件**：
    *   点击左上角的 **"加载已解压的扩展程序" (Load unpacked)** 按钮。
    *   选择第 1 步中解压出来的文件夹（**注意**：是包含 `manifest.json` 的那个文件夹）。

5.  **完成！**
    *   现在打开任意知乎问题页（如 [这个问题](https://www.zhihu.com/question/31599637)），你应该能看到页面瞬间变清爽了。

## 📖 使用说明

1.  **自动启用**：进入知乎的问题详情页或文章页时，插件会自动运行。
2.  **自定义设置**：
    *   点击浏览器右上角的插件图标（那个蓝色的 **Z**）。
    *   *提示：如果没有看到图标，请点击拼图形状的“扩展程序”按钮，将 Zhihu Pure Reader 固定到工具栏。*
    *   在弹出的菜单中，你可以随意切换主题、字体和调整排版参数。
    *   所有设置会自动保存，下次打开自动应用。

## 🛠️ 开发与构建

如果你想修改代码：

*   `manifest.json`: 插件的配置文件。
*   `content.js`: 核心逻辑，负责页面清理和样式注入。
*   `styles.css`: 样式表，控制阅读模式的视觉效果。
*   `popup.html` / `popup.js`: 点击图标后弹出的控制面板。

修改文件后，回到 `chrome://extensions/` 页面点击刷新按钮即可查看效果。
